# Game Popularity Analysis

## プロジェクト概要
ゲームデータを分析し、人気ゲームの傾向を調べました。
PythonとPandasを使ってデータを集計し、Matplotlibで可視化しました。

## 使用ツール
- Python
- Pandas
- Matplotlib

## データ
- game : ゲーム名
- genre : ジャンル
- players : プレイヤー数
- release_year : 発売年

## 分析内容
1. 人気ゲームランキング
2. ジャンル別プレイヤー数
3. ジャンル別シェア分析
4. リリース年別プレイヤー数
5. リリース年別ゲーム数

## 分析結果

・プレイヤー数ランキングでは、Minecraft と League of Legends が最も多かった  
・ジャンル別では、Sandbox と MOBA が最も高かった  
・Battle Royale も大きなシェアを持っていた  
・上位3タイトルに人気が集中している傾向が確認できた  

## 考察

・継続的にプレイされるゲームはプレイヤー数が大きくなりやすいと考えられる  
・対戦型や長期運営型タイトルが上位に入っており、継続プレイ性が人気に影響している可能性がある  
・配信タイトルを考える際には、プレイヤー数の多さだけでなく競争の激しさも考慮する必要がある  

## 分析の限界

・今回のデータは少数のタイトルで構成されているため、市場全体を完全に代表するものではない  
・プレイヤー数以外の指標は含まれていない  
・今後はより大規模な公開データを使って分析を拡張したい
