# A/B Test Analysis

## Overview
This project analyzes the performance of two marketing campaigns (A and B).

## Method
A t-test was conducted to determine if there is a statistically significant difference in conversion rates.

## Result

Campaign B showed a higher conversion rate (70%) compared to Campaign A (30%).

However, the statistical test resulted in a p-value of 0.08, which is above the typical significance threshold (0.05). Therefore, the difference is not statistically significant.
